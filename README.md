# Travel Express Bus Reservation System

A C++ and MySQL-based bus reservation system developed as an academic project.

## Features
- Bus ticket reservation workflow
- Basic CRUD operations using MySQL
- User-friendly console-based interface

## Tech Stack
- C++
- MySQL

## Project Structure
- `src/` - C++ source code
- `database/` - Database scripts (schema)
- `docs/` - Proposal, final report, analysis & design documents
- `slides/` - Presentation slides


## How to Run
1. Import the database script:
   - `database/schema.sql`
2. Compile and run the C++ application:
   - Build `src/main.cpp` using your preferred compiler/IDE.
